## felles-feil

Denne modulen inneholder api og implementasjon for deklarerte feil
og exceptions.


## Hensikten

* Oppnå en helhetlig feil- og exception-håndtering.
* Oppnå at alle logginnslag av alvorlighet ERROR og WARN dokumenteres.
